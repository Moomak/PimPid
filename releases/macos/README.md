# PimPid.app

แอปที่ build แล้วสำหรับ macOS (Apple Silicon) — ไม่ต้อง build เอง

1. ดาวน์โหลดโฟลเดอร์ **PimPid.app**
2. ย้ายไปที่ Applications หรือเปิดดับเบิลคลิกได้เลย
3. ครั้งแรก: เปิด **System Settings → Privacy & Security → Accessibility** แล้วเพิ่ม PimPid และเปิดใช้งาน
4. กด **⌘⇧L** หลังเลือกข้อความที่ผิดภาษาเพื่อแปลง

**หมายเหตุ:** Build จาก `swift build -c release` บน macOS 14+ (arm64) ถ้าต้องการ build เองดูที่ [README หลัก](../README.md)
